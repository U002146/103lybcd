使用說明書

簡單幾步，解決定位與卡頓問題

圖文選單沒更新？

強制重新載入最新選單的隱藏密技：

  1. 在我們的 LINE 聊天室畫面
  2. 點擊右上角的 ≡ (選單)
  3. 點擊 封鎖，再點擊一次 解除封鎖
  4. 按左上角 ＜ 返回 上一頁
  5. 再按 ＜ 返回 聊天分頁
  6. 重新進入聊天室，就會顯示最新選單囉！
定位不準確？

請檢查您的 LINE 是否有開啟定位權限：

iOS (iPhone)
  1. 進入手機的 設定 ⚙️
  2. 滑到最下方找到 LINE
  3. 點選 位置
  4. 選擇 「使用 App 期間」 或 「永遠」
Android (安卓)
  1. 進入手機的 設定 ⚙️
  2. 點選 應用程式 > 找到 LINE
  3. 點選 權限 > 位置
  4. 選擇 「僅在使用該應用程式時允許」
畫面卡住？清除快取
  1. 回到 LINE 的 主頁 🏠 (左下角)
  2. 點擊右上角 設定 ⚙️ (齒輪)
  3. 往下滑找到 聊天
  4. 點選 刪除資料
  5. 只勾選「快取資料」 (不要勾到照片喔！)
  6. 按下 「刪除所選資料」
身分錯誤？重設連動
  1. 回到 LINE 的 主頁 🏠
  2. 點擊右上角 設定 ⚙️
  3. 點選 我的帳號
  4. 點選 連動中的應用程式
  5. 找到 「個人車行 劉學銘 TDP-1123」
  6. 點進去選擇 解除連動
  7. 完成後，重新點擊叫車連結即可！
放在桌面 (安卓限定)
  1. 在我們的 LINE 聊天室畫面
  2. 點擊右上角的 ≡ (選單)
  3. 點擊 其他設定 ⚙️
  4. 選擇 建立聊天室的捷徑
  5. 回到桌面，就會看到圖示囉！
